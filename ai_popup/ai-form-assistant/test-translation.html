üîß Troubleshooting

  • No popup appears: Check browser console (F12) for errors
  • Translation fails: Ensure backend server is running on localhost:8000
  • CORS errors: Make sure you've updated the main.py with the custom CORS middleware
  • Extension not working: Reload the extension in chrome://extensions/
